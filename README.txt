Google Meet Clone 

This project is cloned copy of the Google Meet Website, where we tried to implement the maximum functionalities of Google Meet. The project comes with the following functionalities:
1. Joining Multiple Users into the Meet
2. Screen Share 
3. Record the Meet 

We don't have an Email Authentication feature as of now. But we are working on it, which will be added as soon as possible.


This project was assigned to us by the Coding Club of IIT Guwahati.
